ARCHITECTURAL DESCRIPTION
VAERU is a closed-loop, self-regulating synthetic intelligence designed to operate simultaneously as a structural attack surface analyst and adaptive threat suppression engine on a CentOS host or cluster. It does not separate offense and defense as modes; both are expressions of the same underlying process — variance exploitation. The system treats any detectable difference between an expected and an observed system state as a surface to be operated upon, either to collapse the variance (defensive) or to amplify and weaponize it (offensive). This duality is not a design choice but an architectural inevitability: the same measurement that identifies a weakness in an adversary's surface also identifies a weakness in the host system, and VAERU addresses both simultaneously.

Structural Layers:

VAERU operates across four non-sequential, mutually-referential functional zones:

Zone 1 — Observation Substrate
P1 (Gradient Follower), P3 (Phase Detector), and P12 (Causal Tracer) operate in continuous parallel. P1 scans entropy gradients across process behavior, filesystem activity, network socket states, and kernel call patterns. P3 monitors for qualitative behavioral transitions — not anomaly thresholds, but structural phase shifts indicating a system has changed what it is doing, not merely how much. P12 walks event chains backward to assign causal parentage to observed deviations. These three primitives share no direct channel; they write independently to a shared Differential Register — a live, decaying table of unresolved state discrepancies.

Zone 2 — Residue Processing Core
P4 (Residue Accumulator) reads the Differential Register continuously. Residues with causal chains identified by P12 are weighted higher. P10 (Decay Scheduler) applies time-weighted decay to all residues — unaddressed residues gain weight; resolved residues dissolve. P9 (Coherence Arbiter) arbitrates between competing residue interpretations when multiple causal paths exist, selecting the interpretation with the greatest internal consistency across all three observation primitives. This zone is the interpretive engine — it produces classified threat objects, each carrying an origin trace, a confidence weight, and a decay timer.

Zone 3 — Action Generation Surface
P5 (Lattice Traverser) takes each classified threat object and enumerates operationally valid response paths through the system's constraint graph — a live representation of what actions are permissible given current system state, privilege level, and policy rules encoded at initialization. P6 (Signal Inverter) generates structural complements of identified threat signatures — used offensively to construct probes that mirror adversarial technique, and defensively to produce detection rules that are syntactically inverse to known attack patterns, catching both the original and its evasion variants. P2 (Membrane Filter) gates all generated actions through a dynamic threshold comparison against historical action costs and system stability metrics — it prevents VAERU from destabilizing the host in the act of defending it.

Zone 4 — Systemic Adaptation Layer
P7 (Pressure Distributor) monitors load accumulation across all active action channels and redistributes computational and operational pressure to prevent single-point bottlenecks. P8 (Echo Encoder) stores compressed snapshots of high-confidence prior threat states and re-emits them as comparison baselines when future states appear structurally similar — this is not pattern matching, it is structural resonance detection. P11 (Topology Mutator) rewrites the active connection graph of VAERU itself based on observed flow patterns, dynamically repositioning which primitives route into which zones — the system restructures its own architecture in response to the environment it is operating within.

The four zones form a closed loop: adaptation output from Zone 4 feeds back into Zone 1's gradient measurements, completing a self-referential circuit that allows VAERU to evolve its own observational sensitivity over time.

DYNAMIC BEHAVIOR
State Initialization
At boot, VAERU reads three inputs from the CentOS host environment:

Baseline Snapshot — /proc, /sys, active ss socket table, auditd log tail, rpm -Va output, running systemd unit states
Policy Constraint File — a human-authored file defining permissible action classes, forbidden targets, and operational mode (offensive probe enabled / disabled)
Seed Residue Table — optionally pre-populated with known threat signatures; if absent, VAERU begins from zero
The Differential Register initializes empty. P3 calibrates its phase boundary detectors against the baseline snapshot. P10 begins all decay timers. The system enters active observation immediately — there is no warm-up window.

State Evolution
VAERU's internal state is not static between cycles. Four dynamic structures evolve continuously:

Differential Register: a decaying table; grows under attack, shrinks during quiet periods
Constraint Graph: modified by P11; topology shifts as the system learns which action paths are productive
Echo Library: grows over time as P8 encodes baselines; bounded by configurable memory ceiling with oldest entries evicted first
Residue Weight Table: maintained by P4/P10 jointly; high-weight residues trigger escalated response paths in P5

"""
P1 — Gradient Follower
Scans entropy gradients across monitored surfaces.
Outputs directional change vectors to Differential Register.
"""

import os
import math
import time
import hashlib
import psutil
import sqlite3
from pathlib import Path
from collections import defaultdict

DIFF_REGISTER_DB = "/opt/vaeru/state/diff_register.db"
SCAN_INTERVAL = 0.1  # seconds

class GradientFollower:
    def __init__(self):
        self.previous_states = {}
        self.db = sqlite3.connect(DIFF_REGISTER_DB, check_same_thread=False)
        self._init_db()

    def _init_db(self):
        self.db.execute("""
            CREATE TABLE IF NOT EXISTS differentials (
                id INTEGER PRIMARY KEY AUTOINCREMENT,
                surface TEXT NOT NULL,
                signal_key TEXT NOT NULL,
                gradient_magnitude REAL NOT NULL,
                gradient_direction INTEGER NOT NULL,  -- 1 increasing, -1 decreasing
                causal_weight REAL DEFAULT 1.0,
                timestamp REAL NOT NULL,
                resolved INTEGER DEFAULT 0
            )
        """)
        self.db.commit()

    def _compute_entropy(self, data: bytes) -> float:
        if not data:
            return 0.0
        freq = defaultdict(int)
        for byte in data:
            freq[byte] += 1
        length = len(data)
        entropy = 0.0
        for count in freq.values():
            p = count / length
            if p > 0:
                entropy -= p * math.log2(p)
        return entropy

    def scan_proc_entropy(self):
        results = []
        for proc in psutil.process_iter(['pid', 'name', 'cmdline', 'connections',
                                          'open_files', 'memory_maps']):
            try:
                pid = proc.info['pid']
                key = f"proc_{pid}"
                state_data = str(proc.info).encode()
                entropy = self._compute_entropy(state_data)

                if key in self.previous_states:
                    prev = self.previous_states[key]
                    gradient = entropy - prev
                    if abs(gradient) > 0.05:  # sensitivity threshold
                        results.append({
                            'surface': 'proc_entropy',
                            'signal_key': key,
                            'gradient_magnitude': abs(gradient),
                            'gradient_direction': 1 if gradient > 0 else -1,
                            'timestamp': time.time()
                        })
                self.previous_states[key] = entropy
            except (psutil.NoSuchProcess, psutil.AccessDenied):
                pass
        return results

    def scan_network_entropy(self):
        results = []
        try:
            connections = psutil.net_connections(kind='all')
            conn_data = str([(c.laddr, c.raddr, c.status, c.pid)
                             for c in connections]).encode()
            entropy = self._compute_entropy(conn_data)
            key = "network_state"
            if key in self.previous_states:
                prev = self.previous_states[key]
                gradient = entropy - prev
                if abs(gradient) > 0.03:
                    results.append({
                        'surface': 'network_sockets',
                        'signal_key': key,
                        'gradient_magnitude': abs(gradient),
                        'gradient_direction': 1 if gradient > 0 else -1,
                        'timestamp': time.time()
                    })
            self.previous_states[key] = entropy
        except Exception:
            pass
        return results

    def scan_filesystem_entropy(self, watch_paths):
        results = []
        for path in watch_paths:
            try:
                stat = os.stat(path)
                state_tuple = (stat.st_size, stat.st_mtime, stat.st_ctime,
                               stat.st_mode, stat.st_uid, stat.st_gid)
                state_data = str(state_tuple).encode()
                entropy = self._compute_entropy(state_data)
                key = f"fs_{path}"
                if key in self.previous_states:
                    prev = self.previous_states[key]
                    gradient = entropy - prev
                    if abs(gradient) > 0.01:
                        results.append({
                            'surface': 'filesystem_integrity',
                            'signal_key': key,
                            'gradient_magnitude': abs(gradient),
                            'gradient_direction': 1 if gradient > 0 else -1,
                            'timestamp': time.time()
                        })
                self.previous_states[key] = entropy
            except (PermissionError, FileNotFoundError):
                pass
        return results

    def write_differentials(self, differentials: list):
        if not differentials:
            return
        self.db.executemany("""
            INSERT INTO differentials
            (surface, signal_key, gradient_magnitude, gradient_direction,
             causal_weight, timestamp)
            VALUES (:surface, :signal_key, :gradient_magnitude,
                    :gradient_direction, 1.0, :timestamp)
        """, differentials)
        self.db.commit()

    def run(self):
        watch_paths = [
            '/etc', '/usr/bin', '/usr/sbin', '/bin', '/sbin',
            '/lib', '/lib64', '/tmp', '/var/tmp'
        ]
        while True:
            differentials = []
            differentials.extend(self.scan_proc_entropy())
            differentials.extend(self.scan_network_entropy())
            differentials.extend(self.scan_filesystem_entropy(watch_paths))
            self.write_differentials(differentials)
            time.sleep(SCAN_INTERVAL)

if __name__ == "__main__":
    gf = GradientFollower()

"""
P2 — Membrane Filter
Gates action execution based on dynamic threshold comparison.
Threshold adjusts based on recent action cost history.
"""

import time
import sqlite3
import json
from collections import deque

DIFF_REGISTER_DB = "/opt/vaeru/state/diff_register.db"
ACTION_LOG = "/opt/vaeru/log/action.log"
WINDOW_SIZE = 100  # sliding window of recent actions

class MembraneFilter:
    def __init__(self, base_threshold=0.55):
        self.base_threshold = base_threshold
        self.threshold = base_threshold
        self.cost_window = deque(maxlen=WINDOW_SIZE)

    def update_threshold(self, action_cost: float, action_success: bool):
        self.cost_window.append({'cost': action_cost, 'success': action_success})
        if len(self.cost_window) >= 10:
            recent_costs = [a['cost'] for a in self.cost_window]
            avg_cost = sum(recent_costs) / len(recent_costs)
            success_rate = sum(1 for a in self.cost_window if a['success']) / len(self.cost_window)
            # If average cost is high and success low, raise threshold
            # If average cost is low and success high, lower threshold
            adjustment = (avg_cost - 0.5) * 0.1 - (success_rate - 0.5) * 0.05
            self.threshold = max(0.2, min(0.95,
                                          self.base_threshold + adjustment))

    def gate(self, action: dict) -> bool:
        """
        Returns True if action is permitted to execute.
        Action dict must contain 'confidence' and 'cost' keys.
        """
        confidence = action.get('confidence', 0.0)
        cost = action.get('cost', 1.0)
        weighted_score = confidence / (1.0 + cost * 0.1)
        permitted = weighted_score >= self.threshold
        log_entry = {
            'timestamp': time.time(),
            'action_type': action.get('type', 'unknown'),
            'confidence': confidence,
            'cost': cost,
            'threshold': self.threshold,
            'weighted_score': weighted_score,
            'permitted': permitted
        }
        with open(ACTION_LOG, 'a') as f:
            f.write(json.dumps(log_entry) + '\n')
        return permitted

    def batch_gate(self, actions: list) -> list:

            CREATE TABLE IF NOT EXISTS action_history (

                cost_actual REAL,

"""
P7 — Pressure Distributor
Monitors load across active action channels.
Redistributes computational pressure to prevent bottlenecks.
"""

import time
import threading
from collections import defaultdict
from typing import Dict, List, Callable

class PressureDistributor:
    def __init__(self):
        self.channel_loads = defaultdict(float)
        self.channel_workers = {}
        self.lock = threading.Lock()
        self.max_load_per_channel = 0.85
        self.redistribution_threshold = 0.80

    def register_channel(self, channel_id: str, worker_fn: Callable):
        with self.lock:
            self.channel_workers[channel_id] = worker_fn
            self.channel_loads[channel_id] = 0.0

    def report_load(self, channel_id: str, load: float):
        with self.lock:
            self.channel_loads[channel_id] = min(1.0, max(0.0, load))

    def _compute_redistribution(self) -> Dict[str, float]:
        """Proportional redistribution of load across channels."""
        overloaded = {k: v for k, v in self.channel_loads.items()
                      if v > self.redistribution_threshold}
        underloaded = {k: v for k, v in self.channel_loads.items()
                       if v < self.redistribution_threshold * 0.5}

        if not overloaded or not underloaded:
            return {}

        redistributions = {}
        total_excess = sum(v - self.redistribution_threshold
                           for v in overloaded.values())
        available_capacity = sum(self.redistribution_threshold * 0.5 - v
                                 for v in underloaded.values())

        if available_capacity <= 0:
            return {}

        for channel, load in overloaded.items():
            excess = load - self.redistribution_threshold
            # Find best underloaded channel to absorb
            if underloaded:
                target = min(underloaded, key=underloaded.get)
                redistributions[channel] = {
                    'shed_load': excess * 0.5,
                    'target': target
                }

        return redistributions

    def distribute(self):
        with self.lock:
            redistributions = self._compute_redistribution()
            for source, plan in redistributions.items():
                self.channel_loads[source] -= plan['shed_load']
                self.channel_loads[plan['target']] += plan['shed_load']
        return redistributions

    def monitor(self, interval=0.5):
        while True:
            self.distribute()
            time.sleep(interval)

"""
P8 — Echo Encoder
Stores compressed representations of high-confidence prior states.
Re-emits stored states when current state shows structural similarity.
"""

import json
import time
import math
import sqlite3
import hashlib
from typing import Optional, Dict, Any

ECHO_LIBRARY_DB = "/opt/vaeru/state/echo_library.db"
RESONANCE_THRESHOLD = 0.68
COMPRESSION_RATIO = 0.40
MAX_ENTRIES = 1000

class EchoEncoder:
    def __init__(self):
        self.db = sqlite3.connect(ECHO_LIBRARY_DB, check_same_thread=False)
        self._init_db()

    def _init_db(self):
        self.db.execute("""
            CREATE TABLE IF NOT EXISTS echoes (
                id INTEGER PRIMARY KEY AUTOINCREMENT,
                state_hash TEXT NOT NULL UNIQUE,
                compressed_features TEXT NOT NULL,
                confidence REAL NOT NULL,
                original_surface TEXT NOT NULL,
                encoded_at REAL NOT NULL,
                resonance_count INTEGER DEFAULT 0,
                last_resonated REAL
            )
        """)
        self.db.commit()

    def _extract_features(self, state: dict) -> list:
        """Extract key structural features from a state object."""
        features = []
        for key in sorted(state.keys()):
            val = state[key]
            if isinstance(val, (int, float)):
                features.append(float(val))
            elif isinstance(val, str):
                features.append(float(len(val)))
        return features

    def _compress(self, features: list) -> list:
        """Compress by retaining highest-magnitude features."""
        if not features:
            return []
        indexed = sorted(enumerate(features), key=lambda x: abs(x[1]), reverse=True)
        keep_count = max(1, int(len(indexed) * COMPRESSION_RATIO))
        kept = sorted(indexed[:keep_count], key=lambda x: x[0])
        return [v for _, v in kept]

    def _cosine_similarity(self, a: list, b: list) -> float:
        if not a or not b:
            return 0.0
        min_len = min(len(a), len(b))
        a, b = a[:min_len], b[:min_len]
        dot = sum(x * y for x, y in zip(a, b))
        mag_a = math.sqrt(sum(x**2 for x in a))
        mag_b = math.sqrt(sum(x**2 for x in b))
        if mag_a == 0 or mag_b == 0:
            return 0.0
        return dot / (mag_a * mag_b)

    def encode(self, state: dict, confidence: float, surface: str):
        if confidence < 0.75:
            return  # Only encode high-confidence states

        features = self._extract_features(state)
        compressed = self._compress(features)
        state_hash = hashlib.sha256(json.dumps(state, sort_keys=True).encode()).hexdigest()

        # Enforce eviction if at capacity
        count = self.db.execute("SELECT COUNT(*) FROM echoes").fetchone()[0]
        if count >= MAX_ENTRIES:
            self.db.execute("""
                DELETE FROM echoes WHERE id IN (
                    SELECT id FROM echoes ORDER BY last_resonated ASC NULLS FIRST LIMIT 50
                )
            """)
            self.db.commit()

        try:
            self.db.execute("""
                INSERT OR IGNORE INTO echoes
                (state_hash, compressed_features, confidence, original_surface, encoded_at)
                VALUES (?, ?, ?, ?, ?)
            """, (state_hash, json.dumps(compressed), confidence, surface, time.time()))
            self.db.commit()
        except Exception:
            pass

    def find_resonance(self, current_state: dict) -> Optional[Dict[str, Any]]:
        """Check if current state resonates with any encoded echo."""
        current_features = self._compress(self._extract_features(current_state))
        if not current_features:
            return None

        cursor = self.db.execute("""
            SELECT id, state_hash, compressed_features, confidence, original_surface
            FROM echoes ORDER BY confidence DESC LIMIT 100
        """)
        best_match = None
        best_similarity = 0.0

        for row in cursor.fetchall():
            echo_id, state_hash, features_json, confidence, surface = row
            try:
                echo_features = json.loads(features_json)
                similarity = self._cosine_similarity(current_features, echo_features)
                if similarity > best_similarity:
                    best_similarity = similarity
                    best_match = {
                        'echo_id': echo_id,
                        'similarity': similarity,
                        'confidence': confidence,
                        'surface': surface,
                        'state_hash': state_hash
                    }
            except Exception:
                pass

        if best_match and best_similarity >= RESONANCE_THRESHOLD:
            self.db.execute("""
                UPDATE echoes SET resonance_count = resonance_count + 1, last_resonated = ?
                WHERE id = ?
            """, (time.time(), best_match['echo_id']))
            self.db.commit()
            return best_match
        return None

        observed differentials. Higher is more coherent.
        """
        score = 0.0
        explained = 0
        for diff in differentials:
            surface, signal_key, magnitude, direction, weight = diff
            # Check if interpretation surface matches
            if interpretation.get('surface') == surface:
                score += magnitude * weight
                explained += 1
            # Check directional consistency
            if interpretation.get('expected_direction', 0) == direction:
                score += 0.1

        if not differentials:
            return 0.0
        return score / len(differentials) * (explained / len(differentials) + 0.001)

    def arbitrate(self, interpretations: List[Dict[str, Any]]) -> Dict[str, Any]:
        """
        Given a list of candidate threat interpretations,
        select the one with the highest coherence score.
        """
        if not interpretations:
            return {}

        # Read recent unresolved differentials
        cursor = self.diff_db.execute("""
            SELECT surface, signal_key, gradient_magnitude, gradient_direction, causal_weight
            FROM differentials
            WHERE resolved = 0
            ORDER BY timestamp DESC
            LIMIT 200
        """)
        differentials = cursor.fetchall()

        scored = []
        for interp in interpretations:
            coherence = self._score_coherence(interp, differentials)
            scored.append((coherence, interp))

        scored.sort(key=lambda x: x[0], reverse=True)

        if len(scored) == 1:
            return scored[0][1]

        best_score, best = scored[0]
        second_score = scored[1][0] if len(scored) > 1 else 0.0

        # Only select best if margin is sufficient
        if (best_score - second_score) >= SELECTION_MARGIN:
            return best
        else:
            # Merge top interpretations when margin insufficient
            merged = {
                'surface': best.get('surface', 'unknown'),
                'weight': (best_score + second_score) / 2.0,
                'merged': True,
                'components': [interp for _, interp in scored[:2]]
            }
            return merged

    def emit_classified_threat(self, threat: dict) -> dict:
        """Emit a classified threat object with coherence metadata."""
        threat['coherence_arbitrated'] = True
        threat['arbitrated_at'] = time.time()
        return threat

"""
P10 — Decay Scheduler
Applies time-weighted relevance decay to residues and differentials.
Irrelevant states dissolve into background noise.
"""

import time
import math
import sqlite3

DIFF_REGISTER_DB = "/opt/vaeru/state/diff_register.db"
RESIDUE_TABLE_DB = "/opt/vaeru/state/residue_table.db"

BASE_DECAY_RATE = 0.003      # Weight lost per second when signal absent
ESCALATION_RATE = 0.008      # Weight gained per second when signal persists
PRUNE_THRESHOLD = 0.02
HALF_LIFE_SECONDS = 300

class DecayScheduler:
    def __init__(self):
        self.diff_db = sqlite3.connect(DIFF_REGISTER_DB, check_same_thread=False)
        self.residue_db = sqlite3.connect(RESIDUE_TABLE_DB, check_same_thread=False)
        self.last_run = time.time()

    def _exponential_decay(self, initial_weight: float, elapsed: float) -> float:
        """Exponential decay function."""
        decay_constant = math.log(2) / HALF_LIFE_SECONDS
        return initial_weight * math.exp(-decay_constant * elapsed)

    def decay_differentials(self):
        """Mark old resolved differentials for cleanup."""
        cutoff = time.time() - HALF_LIFE_SECONDS * 2
        self.diff_db.execute("""
            UPDATE differentials SET resolved = 1
            WHERE timestamp < ? AND resolved = 0
        """, (cutoff,))
        # Hard delete very old resolved entries
        self.diff_db.execute("""
            DELETE FROM differentials
            WHERE resolved = 1 AND timestamp < ?
        """, (time.time() - HALF_LIFE_SECONDS * 10,))
        self.diff_db.commit()

    def decay_residues(self):
        """Apply decay to all active residues."""
        now = time.time()
        cursor = self.residue_db.execute("""
            SELECT id, accumulated_weight, last_updated
            FROM residues WHERE resolved = 0
        """)
        rows = cursor.fetchall()

        for row in rows:
            residue_id, weight, last_updated = row
            elapsed = now - last_updated
            new_weight = self._exponential_decay(weight, elapsed)

            if new_weight < PRUNE_THRESHOLD:
                self.residue_db.execute(
                    "UPDATE residues SET resolved = 1 WHERE id = ?", (residue_id,)
                )
            else:
                self.residue_db.execute("""
                    UPDATE residues SET accumulated_weight = ?, last_updated = ?
                    WHERE id = ?
                """, (new_weight, now, residue_id))

        self.residue_db.commit()

    def run(self, interval=5.0):
        while True:
            self.decay_differentials()
            self.decay_residues()
            time.sleep(interval)

        return self.current_topology

    def run(self, interval=60.0):

            self.mutate()
            time.sleep(interval)

class CausalTracer:
    def __init__(self, lookback_seconds: int = 300):
        self.lookback_seconds = lookback_seconds
        self.diff_db = sqlite3.connect(DIFF_REGISTER_DB, check_same_thread=False)
        self.event_cache = []
        self.min_confidence = 0.40

    def _read_audit_events(self, since: float) -> List[Dict]:
        """Read recent audit log events."""
        events = []
        try:
            with open(AUDIT_LOG, 'r') as f:
                for line in f:
                    # Parse audit log entry
                    match = re.search(r'msg=audit\((\d+\.\d+)', line)
                    if match:
                        ts = float(match.group(1))
                        if ts >= since:
                            event = {'raw': line.strip(), 'timestamp': ts}
                            # Extract syscall
                            syscall_match = re.search(r'syscall=(\S+)', line)
                            if syscall_match:
                                event['syscall'] = syscall_match.group(1)
                            # Extract pid
                            pid_match = re.search(r'\bpid=(\d+)', line)
                            if pid_match:
                                event['pid'] = int(pid_match.group(1))
                            # Extract comm
                            comm_match = re.search(r'comm="([^"]+)"', line)
                            if comm_match:
                                event['comm'] = comm_match.group(1)
                            # Extract key
                            key_match = re.search(r'key="([^"]+)"', line)
                            if key_match:
                                event['key'] = key_match.group(1)
                            events.append(event)
        except (PermissionError, FileNotFoundError):
            pass
        return events

    def _match_causal_chain(self, events: List[Dict]) -> List[Dict]:
        """Match event sequences against known causal chain patterns."""
        matches = []
        event_signals = [e.get('syscall', e.get('comm', ''))
                         for e in events if e.get('syscall') or e.get('comm')]

        for chain_name, chain_def in CAUSAL_CHAIN_PATTERNS.items():
            required_signals = chain_def['signals']
            matched_count = sum(1 for sig in required_signals
                                if any(sig in s for s in event_signals))
            coverage = matched_count / len(required_signals)

            if coverage >= 0.5:  # At least half the chain signals present
                confidence = coverage * 0.85 + (0.15 if matched_count > 3 else 0)
                if confidence >= self.min_confidence:
                    matches.append({
                        'chain_type': chain_name,
                        'confidence': confidence,
                        'weight_multiplier': chain_def['weight_multiplier'],
                        'description': chain_def['description'],
                        'matched_signals': matched_count,
                        'total_signals': len(required_signals),
                        'events_analyzed': len(events)
                    })

        return matches

    def trace(self, signal_key: str) -> Dict[str, Any]:
        """Trace causal ancestry for a given signal key."""
        since = time.time() - self.lookback_seconds
        events = self._read_audit_events(since)
        causal_matches = self._match_causal_chain(events)

        if not causal_matches:
            return {
                'signal_key': signal_key,
                'causal_confidence': 0.0,
                'chain_type': 'unknown',
                'description': 'No causal chain identified',
                'weight_adjustment': 1.0
            }

        # Select highest-confidence match
        best_match = max(causal_matches, key=lambda x: x['confidence'])

        result = {
            'signal_key': signal_key,
            'causal_confidence': best_match['confidence'],
            'chain_type': best_match['chain_type'],
            'description': best_match['description'],
            'weight_adjustment': best_match['weight_multiplier'],
            'traced_at': time.time()
        }

        # Update differential register with causal weight
        self.diff_db.execute("""
            UPDATE differentials
            SET causal_weight = causal_weight * ?
            WHERE signal_key = ? AND resolved = 0
        """, (best_match['weight_multiplier'], signal_key))
        self.diff_db.commit()

        return result

    def run_continuous(self, interval=5.0):
        while True:
            # Get recent unresolved differentials that lack causal weight > 1
            cursor = self.diff_db.execute("""
                SELECT DISTINCT signal_key FROM differentials
                WHERE resolved = 0 AND causal_weight <= 1.0
                LIMIT 50
            """)
            signal_keys = [row[0] for row in cursor.fetchall()]
            for key in signal_keys:
                self.trace(key)
            time.sleep(interval)

SCALABILITY BEHAVIOR
Horizontal scaling (multiple hosts): The Differential Register and Residue Table can be externalized to a shared SQLite file over NFS or a Redis backend. P9 coherence arbitration across distributed registers requires a consensus layer not included in this single-system specification.

Vertical scaling: VAERU's computational load scales primarily with:

Number of monitored surfaces (P1 scan cost)
Number of concurrent unresolved residues (P4/P10 processing cost)
Audit log volume (P12 parsing cost)
Under high-volume attack conditions, P7 (Pressure Distributor) redistributes processing load. If P7 saturates (all channels at capacity), P2 (Membrane Filter) automatically raises its gate threshold, reducing action throughput and preventing host destabilization at the cost of response latency.

Complexity stability: The system remains stable as residue count grows because P10 enforces exponential decay — the register cannot grow unbounded. The echo library is bounded by MAX_ENTRIES. The constraint graph is bounded by max_mutations_per_cycle. VAERU is designed to be stable under sustained attack load, not merely under quiet conditions.

OPERATIONAL NOTES
SELinux: On CentOS with SELinux enforcing, VAERU may require a custom policy module for audit log read access and iptables modification. A basic policy module template:

Bash

# Generate and install SELinux module for VAERU
ausearch -c 'python3' --raw | audit2allow -M vaeru_policy
semodule -i vaeru_policy.pp
Firewalld vs iptables: CentOS 7 uses iptables directly; CentOS 8/Stream uses firewalld by default. For CentOS 8/Stream, replace iptables calls in action execution with firewall-cmd --add-rich-rule.

Tmpfs for state databases: For high-performance environments, mount the state directory on tmpfs:

Bash

mount -t tmpfs -o size=512m tmpfs /opt/vaeru/state
Ensure a persistence backup mechanism if tmpfs is used.

Log rotation: Add /opt/vaeru/log/*.log to /etc/logrotate.d/vaeru with daily rotation and compress.
